About:

This extension is the result of a Google Summer of Code 2012 project by Zachariah Moreno with mentors John J Barton, John Arthorne, & Ken Walker. It is an implimentation of the Eclipse Orion editor into a Google Chrome DevTools extension with the goal of enhancing the current development workflow by bringing it to the browser.
