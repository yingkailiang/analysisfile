Markdown Hereを使うことで、電子メールをMarkdownで書いて送信前に (かっこよく!) レンダリングすることができます。

電子メールを書いている時にメールの書式をボタンでポチポチいじることが嫌いな人にとっては、Markdown Hereは最高の拡張機能です。 構文のハイライトだってサポートしています。メールの中にコードを書きたいプログラマーの方にとっても最高です。 また、数学をする人のために、TeXで書かれた数式もレンダリングします。

使い方はとっても簡単です:
1. 電子メールをリッチなエディター上で、GitHub-flavorated Markdownで書きます。
2. 右クリックメニューで「Markdown 切り替え」をクリックします。
3. あなたの電子メールはキレイに整形されています! (つまり、HTMLとしてレンダリングされます)
4. 見た目が気に入ったなら、メールを送信するだけ。 もし編集したりなにかを追加したかったら、もう一度「Markdown 切り替え」をクリックすると、元のMarkdownに戻すことができます。 練り直しましょう。

Markdown HereはGmailとThunderbirdを主にターゲットとしていますが、YahooやHotmailでもよく動作します。 また、Google グループやGoogle サイト、Evernote、Blogger、Wordpress、もっといろんなサービスでも動作します！

コードのハイライト表示について: 「fenced block」(```) で囲み、言語の名前を指定することで使用できます。 プロジェクトのページにある例もご覧ください。

プライバシー:
Markdown Hereはあなたのメールの内容にアクセスし、変更します。 理論的にはその他のwebコンテンツにもアクセスできますが、行いません。 また、インターネットでの通信も一切行いません。 あなたのデータはあなたが選んだ場所・時間にしか変更されず、ブラウザーにデータを残すこともありません。

Chrome、Firefox、Safari、Opera、Thunderbirdの拡張機能として使用することができます。

Markdown Hereはオープンソースです。 完全な使い方を見たり、詳細を参照したり、バグの報告、コントリビュートをするには、Githubにあるプロジェクトページを訪れてください。
